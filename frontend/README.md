Campus Resource Hub is a full-stack MERN web application designed to help students easily discover, add, and manage important campus-related resources such as academic portals, placement updates, and wellbeing support.

---

##  Features

- User authentication (Register & Login) using JWT
- Secure and protected routes
- Add new campus resources
- Edit and delete existing resources
- Search resources by title and description
- Filter resources by category (Academics, Placements, Wellbeing)
- Dashboard statistics showing total and category-wise resources
- Responsive and clean UI
